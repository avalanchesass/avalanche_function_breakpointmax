# avalanchesass_function_breakpointmax
Get the max-width value of a breakpoint.

## Install
```
npm install avalanchesass_function_breakpointmax --save
```

## Basic usage
```css
/* Import the main file. */
@import 'avalanchesass_function_breakpointmax';
```
